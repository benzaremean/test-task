## Installation

## Prerequisites
You need to install the following
1. Node js
2. Java JDK 1.8
3. Firefox 22

To install run following command from root folder

``` shell
$ npm install
```

## Tests

To run all Tests
``` shell
$ npm run test
```
To run UI Tests
``` shell
$ npm run ui-test
```
To run API Tests
``` shell
$ npm run api-test
```
